# 奔跑羚羊-电阻网络专家 (Running Antelope Resistor Expert)

这是一个基于 Python 和 Tkinter 开发的专业级电阻网络计算工具，专为电子工程师设计。它不仅支持标准的分压计算，还深度支持复杂的串并联混合网络、NTC 热敏电阻应用、功率分配分析以及电路拓扑可视化。

## ✨ 主要功能

*   **复杂电阻网络支持**：
    > **Complex Network Support**: Easily handle arbitrary series and parallel combinations for R1/R2, including nested structures.
    *   支持 R1 (上拉) 和 R2 (下拉) 的任意串联和并联组合。
    *   支持嵌套并联组，方便通过多电阻并联凑出非标准阻值。
*   **可视化电路拓扑**：
    *   实时绘制电路拓扑图，直观显示电阻连接方式。
    *   窗口大小自适应，支持动态重绘。
    *   智能显示关键节点电压 (Vin, Vout) 和等效阻值。
    *   安全状态指示（根据 Vout 自动判断是否过压）。
*   **应用场景模板**：
    *   内置多种典型应用场景（如电池电压监测、电平转换、NTC 测温等）。
    *   一键加载预设参数，快速开始设计。
*   **NTC 热敏电阻支持**：
    *   内置常用 NTC 型号参数 (MF52, MF58 等)。
    *   支持自定义 NTC 参数 (R25, B值)。
    *   提供独立的 NTC 阻值/温度计算器。
*   **辅助工具**：
    *   **计算缺失电阻**：已知 Vout 反推 R1 或 R2。
    *   **推荐标准值**：基于 E24/E96 系列推荐最接近的标准电阻组合。
    *   **并联计算器**：快速计算并联等效阻值。
    *   **功率分配分析**：分析并联电阻的功率分担情况。
    *   **精度优化建议**：提供高精度电阻组合方案。
*   **数据管理**：
    *   支持导出 BOM (物料清单)。
    *   支持保存和加载设计配置 (JSON 格式)。

## 🛠️ 运行环境

*   Python 3.6+
*   标准库依赖：
    *   `tkinter` (GUI 界面)
    *   `math` (数学计算)
    *   `json` (配置存取)
    *   `datetime` (BOM 导出时间)

## 🚀 快速开始

1.  确保已安装 Python 3。
2.  直接运行主程序脚本：

```bash
python resistor_divider_gui.py
```

## 📖 使用指南

### 1. 基础分压设计
*   在左侧面板设置 **Vin** (输入电压) 和 **Vout 目标**。
*   在右侧工作区分别配置 R1 和 R2 网络。
*   可以直接输入阻值，也可以通过“添加串联”或“添加并联组”构建复杂网络。
*   双击列表中的电阻项可进行编辑，选中后点击“删除选中”可移除。

### 2. 使用 NTC 热敏电阻
*   在左侧勾选 **"R2 使用 NTC"**。
*   选择 NTC 型号或输入自定义参数。
*   此时 R2 将作为 NTC 参与分压计算，电路图也会相应更新符号。

### 3. 查看结果
*   底部文本框会实时显示计算结果，包括实际输出电压、误差分析、功耗估算等。
*   底部画布区域会展示当前的电路连接图。
