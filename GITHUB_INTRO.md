# 奔跑羚羊-电阻网络专家 (Running Antelope Resistor Expert)

**[中文介绍]**

**奔跑羚羊-电阻网络专家** 是一款专为电子工程师打造的专业级电阻网络计算与可视化工具。它超越了简单的分压计算，深度支持复杂的串并联混合网络设计，帮助您快速验证电路方案。

**✨ 核心亮点：**
*   **复杂网络支持**：轻松处理 R1/R2 的任意串联与并联组合，支持嵌套结构。
*   **实时拓扑可视化**：动态绘制电路拓扑图，直观展示连接关系与关键节点电压。
*   **NTC 热敏电阻集成**：内置常用 NTC 模型，支持温度/阻值换算与分压模拟。
*   **工程辅助功能**：包含功率分配分析、精度优化建议、E24/E96 标准值推荐及 BOM 导出。
*   **轻量跨平台**：基于 Python + Tkinter 开发，无需复杂安装，即开即用。

---

**[English Introduction]**

**Running Antelope Resistor Expert** is a professional-grade resistor network calculation and visualization tool tailored for electronics engineers. Going beyond simple voltage division, it provides deep support for complex series-parallel hybrid network designs, enabling rapid circuit verification.

**✨ Key Features:**
*   **Complex Network Support**: Easily handle arbitrary series and parallel combinations for R1/R2, including nested structures.
*   **Real-time Topology Visualization**: Dynamically renders circuit topology, visually displaying connections and key node voltages.
*   **NTC Thermistor Integration**: Built-in common NTC models with support for temperature/resistance conversion and divider simulation.
*   **Engineering Utilities**: Includes power distribution analysis, precision optimization suggestions, E24/E96 standard value recommendations, and BOM export.
*   **Lightweight & Cross-Platform**: Built with Python + Tkinter, requiring no complex installation—just run and use.
